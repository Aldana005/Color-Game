# RGB Color Game

Juego web donde debes adivinar el código de color RGB correcto. Desarrollado para practicar lógica de programación y manipulación del DOM.

[Jugar Ahora](https://color-game-sanchez.netlify.app/)

#Características
* 2 Modos:Fácil (3 opciones) y Difícil (6 opciones).
* Responsive:Funciona en celulares y escritorio.
* Tecnología: HTML5, CSS3 y Vanilla JavaScript (Sin librerías).
